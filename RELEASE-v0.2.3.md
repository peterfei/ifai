# IfAI Editor v0.2.3 发布说明

发布日期：2025-12-26

## 📋 版本概述

**v0.2.3** 是 IfAI 自发布以来最大规模的一次功能升级，新增了超过 10,000 行代码！我们重点引入了**工业级任务监控系统**，彻底重构了**代码库探索体验**，并带来了全新的**项目级配置方案**。此外，文件树与搜索功能的深度增强，将开发效率推向了新高度。

## 🚀 核心特性

### 1. 工业级任务监控系统 (Task Monitor System)
我们引入了一套完整的、可视化的后台任务监控体系，让每一个异步操作都清晰可见：
- **实时进度流**：通过 `TaskMonitor` 面板，实时追踪 Agent 思考、文件扫描、RAG 索引构建等任务的每一毫秒进展。
- **可视化指标**：全新的任务仪表盘展示任务成功率、平均耗时、错误率等关键性能指标。
- **日志流分析**：集成 ANSI 彩色日志流，支持实时查看和分析后台任务的详细输出。

### 2. 项目级智能配置 (Project Config)
IfAI 现在支持项目级别的个性化配置：
- **IFAI.md 配置文件**：在项目根目录下创建 `IFAI.md`，即可定义项目专属的 AI 行为规则。
- **自定义指令 (Custom Instructions)**：为不同项目注入特定的 Prompt 上下文（如代码规范、架构约束），AI 会自动遵守。

### 3. 代码库探索 2.0 (Explore Agent 2.0)
彻底重构了代码库探索与理解模块：
- **统一状态管理**：基于 `ExploreStore` 实现扫描状态的全局同步，确保进度条与实际后台任务精确对应。
- **视觉升级**：全新的 `ExploreProgress` 和 `ExploreFindings` 组件，提供更直观的代码分析报告。

### 4. 文件树与搜索增强
- **虚拟滚动文件树**：针对包含数万文件的大型项目，文件树现采用虚拟滚动技术，渲染性能提升 10 倍，滚动如丝般顺滑。
- **全功能右键菜单**：文件树现支持完整的右键菜单操作（新建、重命名、删除、在终端打开等）。
- **实时增强搜索**：搜索面板新增历史记录、结果实时预览及键盘导航支持，查找代码更加得心应手。

### 5. 性能监控工具
- **CacheStatsPanel**：新增缓存命中率监控面板，帮助开发者实时掌握系统资源使用情况。
- **并行扫描优化**：Rust 后端实现了目录并行扫描与增量 Git 状态更新，大幅降低 I/O 延迟。

## 🛠 技术细节与修复

### ✨ 新增组件与模块
- `TaskMonitor`: 核心任务监控容器
- `TaskCard`: 任务状态卡片
- `TaskLogStream`: 实时日志流组件
- `ContextMenu`: 自定义右键菜单系统
- `VirtualFileTree`: 虚拟滚动文件树实现
- `projectConfigStore`: 项目配置状态管理

### 🐛 关键修复
- 修复了新项目未自动创建 `IFAI.md` 模板的问题
- 修复了 Agent 扫描任务完成状态在前端同步延迟的问题
- 统一了不同模块间进度显示的 UI 风格
