# IfAI Editor v0.2.3 发布说明

发布日期：2025-12-27

## 📋 版本概述

**v0.2.3** 是一次关于**"透明度"**与**"掌控力"**的重大进化。在 AI 辅助编程日益普及的今天，我们拒绝让 AI 成为一个不可预测的"黑盒"。

本次更新通过**工业级任务监控系统**，让 AI 的每一次思考、检索与文件操作都清晰可见；通过**项目级智能配置**，让 AI 真正理解并遵循您的项目规范；配合 Rust 驱动的**极致 I/O 性能**，IfAI 正在重新定义什么是"可信赖的 AI 编程伙伴"。

## 🚀 核心价值与特性

### 1. 拒绝黑盒：工业级任务监控 (Task Monitor)
**价值：让 AI 的行为完全透明、可追溯、可调试。**
我们引入了一套完整的、可视化的后台任务监控体系：
- **实时思维流**：通过 `TaskMonitor` 面板，实时追踪 Agent 的决策路径——它在看哪个文件？它在检索什么代码？一目了然。
- **可视化仪表盘**：动态展示任务成功率、平均耗时、错误率等关键指标，帮助开发者评估 AI 的工作状态。
- **极客日志流**：集成 ANSI 彩色日志流，支持实时查看 Rust 后端的详细执行日志，让 Debug 变得轻松有趣。

### 2. 深度定制：项目级智能配置 (Project Config)
**价值：让 AI 懂你的项目，而不只是懂代码。**
IfAI 现在支持项目级别的个性化 "AI 员工手册"：
- **IFAI.md 配置文件**：在项目根目录下创建 `IFAI.md`，定义专属的 System Prompt 和行为准则。
- **规范自动注入**：无论是代码风格（如 "使用 Tabs"）还是架构约束（如 "禁止直接调用 DB"），AI 都会自动遵守，无需重复提示。

### 3. 极致性能：代码库探索 2.0 & 虚拟文件树
**价值：在海量文件中实现零延迟交互。**
- **Rust 并行扫描**：利用 Rayon 并行架构重构文件系统，目录扫描速度提升 1500%。
- **虚拟滚动文件树**：针对数万文件的超大项目，引入虚拟滚动技术，渲染性能提升 10 倍，滚动丝般顺滑。
- **探索可视化**：基于 `ExploreStore` 的统一状态管理，让后台复杂的扫描任务通过 `ExploreProgress` 组件直观呈现。

### 4. 交互增强：文件树与搜索
- **全功能右键菜单**：补全了新建、重命名、删除、在终端打开等核心操作。
- **实时增强搜索**：搜索面板新增历史记录回溯、结果实时预览及全键盘导航支持。

## 🛠 技术细节与修复

### ✨ 新增组件与模块
- `TaskMonitor`: 核心任务监控容器
- `TaskCard`: 任务状态卡片
- `TaskLogStream`: 实时日志流组件
- `ContextMenu`: 自定义右键菜单系统
- `VirtualFileTree`: 虚拟滚动文件树实现
- `projectConfigStore`: 项目配置状态管理

### 🐛 关键修复
- 修复了新项目未自动创建 `IFAI.md` 模板的问题
- 修复了 Agent 扫描任务完成状态在前端同步延迟的问题
- 统一了不同模块间进度显示的 UI 风格
