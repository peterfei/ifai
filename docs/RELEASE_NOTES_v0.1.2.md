# IfAI v0.1.2 Release Notes

## 🚀 新增功能 (New Features)

- **文件拖放支持 (File Drag & Drop)**:
  - 支持文件和文件夹在文件树中的拖拽移动。
  - 支持从系统资源管理器拖入文件到编辑器。
  - 实现了文件拖放的稳定性优化。

- **快捷键系统 (Shortcut System)**:
  - **配置管理**: 支持快捷键配置的导入和导出。
  - **预设方案**: 新增快捷键预设方案切换功能，方便适应不同习惯。

- **窗口与布局增强 (Window & Layout)**:
  - **状态持久化**: 记住窗口大小、位置及布局状态，重启后自动恢复。
  - **多窗口支持**: 实现了多窗口状态的实时同步。
  - **分屏优化**: 改进了分屏体验和标签页 UI 设计。

- **AI 助手增强 (AI Assistant)**:
  - **工具调用闭环**: 完善了多审视流程中的 AI 工具调用。
  - **Diff 预览**: 增强了代码修改的 Diff 预览体验。
  - **品牌化**: 优化了聊天助手的品牌展示。

- **其他**:
  - **国际化 (i18n)**: 文件树右键菜单及设置面板支持多语言。
  - **多光标编辑**: (包含自 v0.1.1) 引入了 Monaco 编辑器的多光标编辑功能。

## 🐛 问题修复 (Bug Fixes)

- **UI/布局**:
  - 修复了打开文件时可能出现空白窗格的问题。
  - 修复了应用重启后内容为空、布局变形及文件名显示错误。
  - 修复了标签页切换时无内容显示及滚动条异常。
  - 修复了布局持久化数据的语法错误。

- **系统集成**:
  - 修复了 MacOS 上应用图标构建失败的问题 (转换为 RGBA 格式)。
  - 修复了 macOS Universal Build 的交叉编译链接错误。
  - 修复了 Vite 热更新错误。

- **其他**:
  - 修复了 Command Palette 打开文件后的窗格关联问题。
