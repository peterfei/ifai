# IfAI v0.2.5 Release Notes

## 概览 (Overview)

IfAI v0.2.5 是一个里程碑式的版本，标志着编辑器正式迈入**"混合智能 (Hybrid Intelligence)"**时代。我们首次引入了完整的本地大语言模型（Local LLM）支持，让 AI 辅助不再完全依赖网络。

## 🚀 核心特性 (Key Features)

### 🔒 本地隐私模型 (Local Privacy-First Models)
- **离线能力**: 内置高性能轻量级模型（基于 Qwen2.5-Coder 优化），无需联网即可享受智能代码补全和基础问答。
- **隐私安全**: 您的代码片段无需上传至云端，敏感项目开发更安心。
- **零延迟响应**: 本地推理消除了网络延迟，打字时的补全反馈快如闪电。

### 🧠 智能路由架构 (Intelligence Router)
- **自动决策**: 系统能够分析您的意图复杂度和当前上下文，自动判断该使用本地模型（快速、免费）还是云端模型（强大、深度）。
- **无缝切换**: 对用户完全透明，您只需专注于编码，IfAI 会在后台为您选择最佳的算力提供方。

### 📦 开箱即用体验
- **一键部署**: 新增引导式模型下载器，自动检测硬件环境并下载最适配的量化模型版本。
- **资源管理**: 智能显存管理，确保在运行模型的同时不影响编辑器的流畅度。

## ✨ 优化与改进 (Improvements)

- **FIM (Fill-In-the-Middle)**: 实现了基于本地模型的光标处中间填充能力，补全不再局限于行尾。
- **启动引导**: 全新的欢迎界面和设置向导，帮助新用户快速配置本地环境。
- **设置面板**: 新增本地模型管理页面，支持手动切换模型路径和参数。

## 🐛 问题修复 (Bug Fixes)

- 修复了在网络不稳定的情况下，工具调用可能会中断的问题。
- 优化了 FileTree 在处理大量文件变动时的刷新逻辑。
